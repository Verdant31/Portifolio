## ⚙️ Installation
```
# Open a terminal and type the command above
$ git clone https://github.com/Verdant31/Portifolio.git
```

```
# Access the application folder
$ cd Portifolio

# Install the dependecies
$ yarn

# Start the application
$ yarn dev

```
